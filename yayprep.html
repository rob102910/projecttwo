What does SQL JOIN do?

JOIN is used to combine rows from two or more columns based on a related column. This can be done between the table and itself, two tables, or multiple tables.

What about INNER JOIN?

Inner JOIN specifically returns the records that have matching values in both tables. This means rows where both tables have information present.

It follows the format 

SELECT column_name(s)
FROM table1
INNER JOIN table2
ON table1.column_name = table2.column_name;

And additional tables may be added by adding the code below. 

INNER JOIN table3
ON table1.column_name = table3.column_name;

What about LEFT JOIN?

Left Join in turn returns all the records from the left table (or your first table specified) along with the corresponding rows from the right table (or your second table), even when no data for the record is present. In this case the record returns the value null.

What about RIGHT JOIN?

Right Join returns all records from the right table (or the second table) and the matched records from the left table (or your first table). It returns null when no data is present for a record of the left table.

What about FULL OUTER JOIN?

Full outer join returns corresponding records when there is a match for either tables record.

What about SELF JOIN?

Self join is the same as a regular join but between a table and itself.
